## End User Resources

### Official Website

[mochajs.org](https://mochajs.org) is the official documentation site for Mocha, providing comprehensive guides, API documentation, and examples.

### Discord Community

Join our active Discord community to ask questions, share experiences, and connect with other Mocha users:

- **Discord Server**: [https://discord.gg/KeDn2uXhER](https://discord.gg/KeDn2uXhER)
- Get help in the `#help` channel
- Discuss features and report bugs
- Connect with maintainers and contributors

### GitHub

The Mocha project is hosted on GitHub under the **mochajs** organization:

- **Main Repository**: [github.com/mochajs/mocha](https://github.com/mochajs/mocha)
- **Organization**: [github.com/mochajs](https://github.com/mochajs)

#### GitHub Resources

- **Issues**: [Report bugs and request features](https://github.com/mochajs/mocha/issues)
- **Discussions**: Coming soon
- **Pull Requests**: [View ongoing contributions](https://github.com/mochajs/mocha/pulls)
- **Releases**: [View release history](https://github.com/mochajs/mocha/releases)

### Stack Overflow

For general questions about using Mocha, you can also use [Stack Overflow](https://stackoverflow.com/questions/tagged/mocha) with the `mocha` tag. While not officially maintained, the community is active there.

### Social Media

Follow Mocha on social platforms for updates and announcements:

- **npm Package**: [npmjs.com/package/mocha](https://www.npmjs.com/package/mocha)
- **GitHub Stars**: Star the repository to show your support and stay updated

## Getting Help

### Before Asking

1. **Search the documentation**: Most common questions are answered in the [docs](https://mochajs.org)
2. **Search existing issues**: Your question might already be answered in [GitHub issues](https://github.com/mochajs/mocha/issues)
3. **Check Stack Overflow**: Look for the `mocha` tag

### Where to Ask

- **Usage questions**: Discord `#help` channel or Stack Overflow
- **Bug reports**: [GitHub Issues](https://github.com/mochajs/mocha/issues/new?template=01-bug.yml)
- **Feature requests**: [GitHub Issues](https://github.com/mochajs/mocha/issues/new?template=03-feature-request.yml)
- **Documentation improvements**: [GitHub Issues](https://github.com/mochajs/mocha/issues/new?template=02-docs.yml)

## Code of Conduct

All community interactions are governed by our [Code of Conduct](https://github.com/mochajs/mocha/blob/main/.github/CODE_OF_CONDUCT.md). Please read and follow it to ensure a welcoming environment for everyone.
