Mocha supports TypeScript out of the box using [Node.js type stripping](https://nodejs.org/api/typescript.html#type-stripping) (available in Node.js 22.6.0+). Type stripping is newly [stable in Node.js as of November 2025](https://github.com/nodejs/node/releases/tag/v25.2.0).

## Quick Start

Install Mocha, Mocha's types, and Node.js types:

<PackageManagers dev type="add" pkg="mocha @types/mocha @types/node" />

Create a test file:

```typescript
// example.test.ts
import * as assert from "node:assert";

describe("Array", function () {
  it("should return -1 when value is not present", function () {
    assert.equal([1, 2, 3].indexOf(4), -1);
  });
});
```

Run your tests:

```bash
npx mocha example.test.ts

# Older versions of Node require an experimental flag
npx mocha --experimental-strip-types example.test.ts
```

In case of IDE issues not recognizing types, you can optionally add an empty `tsconfig.json`:

```js
// tsconfig.json
{
}
```

Upon restarting your IDE, any type errors should go away.
